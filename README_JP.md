# Zynq-7000 チュートリアル

このチュートリアルは、Vivadoでのハードウェア設計とVitisでのソフトウェア開発を含む、Zynq-7000開発の包括的な例を提供します。

## 概要

05_Zynq7000チュートリアルには以下が含まれています：

### ハードウェア設計（Vivado）
- **ブロックデザイン**: ペリフェラルを含む完全なZynq PS設定
- **GPIO設定**: 外部GPIOとEMIOセットアップ
- **ペリフェラル統合**: UART、イーサネット、タイマー、その他ペリフェラル
- **ビットストリーム生成**: FPGAプログラミングファイル

### ソフトウェアアプリケーション（Vitis）
- **LED外部テスト1**: LED制御のための直接GPIOレジスタアクセス
- **LED外部テスト2**: Xilinx GPIOドライバベースのLED制御
- **lwIPエコーサーバー**: lwIPスタックを使用したネットワークエコーサーバー
- **メモリテスト**: 包括的なメモリ検証
- **ペリフェラルテスト**: 複数ペリフェラル機能テスト
- **PS LCD**: PS GPIO経由のLCDディスプレイ制御

## ハードウェア要件

- Zynq-7000ベースの開発ボード（EBAZ4205）
- テストアプリケーション用LED
- ネットワークアプリケーション用イーサネット接続
- PS LCDアプリケーション用LCDディスプレイ
  - [LCD付きアダプタ](https://ja.aliexpress.com/item/1005006074065888.html)（推奨）
- プログラミング/デバッグ用JTAGまたはUART接続

## ソフトウェア要件

- Xilinx Vivado Design Suite
- Xilinx Vitis IDE
- Zynq-7000 BSP（ボードサポートパッケージ）
- 必要なXilinxドライバライブラリ

## はじめに

1. **ハードウェアセットアップ**
   - `vivado/`ディレクトリでVivadoプロジェクトを開く
   - ブロックデザイン設定を確認
   - ビットストリームを生成しVitisにエクスポート

2. **ソフトウェア開発**
   - `vitis/`ディレクトリからVitisワークスペースをインポート
   - 目的のアプリケーションプロジェクトをビルド
   - FPGAをプログラムしアプリケーションを実行

3. **テスト**
   - デバッグのためにUART出力を監視
   - ハードウェア機能を検証
   - アプリケーション固有の機能をテスト

## アプリケーション概要

| アプリケーション | 目的 | 主な機能 |
|----------------|------|----------|
| LED外部テスト1 | 基本GPIO制御 | 直接レジスタアクセス、8状態パターン |
| LED外部テスト2 | ドライバベースGPIO | XGpioドライバ、エラーハンドリング |
| lwIPエコーサーバー | ネットワークテスト | TCP/UDPエコー、IPv4/IPv6サポート |
| メモリテスト | メモリ検証 | 包括的テスト、エラーレポート |
| ペリフェラルテスト | 複数ペリフェラルテスト | GPIO、UART、I2C、SPI、タイマー、DMA |
| PS LCD | ディスプレイ制御 | EMIO GPIO、カラーレンダリング |

## 共通開発ワークフロー

1. **ハードウェア設計**: VivadoでFPGAを設定
2. **ソフトウェア開発**: Vitisでアプリケーションを作成
3. **テスト**: プログラムして機能を検証
4. **デバッグ**: UARTを監視して動作を確認
